The content of this webpage is intended for those who like DIY. I hope you'll all enjoy it one day!

To be successful in DIY you need to have the following qualities:

  • Manual Ability.
  • Tools.
  • Space.
  • Time.

To approach your first DIY project you should proceed as follows:

  1. Sketch what you intend to build.
  2. Check all tools needed are already available.
  3. Define the production process and milestones.
  4. Source all materials and consumables required.
  5. Check online material for refernce.
